# Testing Stack Policy

## Unit

* Runner: Vitest
* Command: `pnpm test:unit`

## E2E

* Runner: Playwright
* Command: `pnpm e2e`

## Selector Rules

* Use `data-testid` selectors.
* Avoid fragile CSS selectors.

## MCP Usage

* Use the `context7` MCP server for Playwright investigations and examples.
